HTML/CSS, What Are They?

    What is HTML?
  • This stands for HyperText Message Language
  • This is used to allow images and objects to be embedded or place on a specific spot on the website and can be use to create interactive forms
  • This was first made by Tim Berners-Lee at 1989 with the purpose of making website easier to make which is also why its consider important
  • This just written in codes with an opening and ending tag and sometimes has text with in those tags
    What is CSS?
    1. This stands for Cascading Style Sheets
    2. This is use for describing the look and formatting of a document written in a markup language, similar to HTML
    3. CSS was created by a man named Hakon Wium Lie around 1994
    What are tags?
  • Better known as HTML tags, they are the hidden keywords within a web page that define how the browser must format and display the content.
  • Almost every tags have two parts, an opening and a closing parts/tags.

Here are a few explained example of HTML tag.

  1. One of the most common tag the use of p, which stands for paragraph in markup language; this mainly use when making a sentence(s).
  2. Another common tag which is actually the one you always start when making a website, the html tag is the second tag you use after the !DOCTYPE html tag. This is tag job is to tell the browser that this is an HTML document and can be put on the internet.
  3. Another tag that's often use is the h1-9 tags. These are uses to differentiate the heading of a page from the rest of the content by making certain phrases/words bigger than other.
  4. And possibly the common and one of the important tag is the body tag which is use to to make sure that page is part of the html is visible to the one seeing it.
